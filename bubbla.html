BUBBLA

Bubbla is an ongoing iOS game project that I've been working on since last summer. Developed in Unity3d, this project is at its core a SpaceShooter game (with modified contents). Check out the videos below to get a general sense of the game!

You can find the github repository here. In the process of developing this game, I worked with Unity colliders/physics system, navigation algorithms, sprites and masking, dependency injection, save/load of files, and constructing systems of code that allow for the most efficient programming of game content.

As an example of simplifying the game dev process, in my project, placing an enemy on a certain spot in the scene can be as simple as specifying several numbers in a csv file. The game parses the file on start and loads in the corresponding coded content into the game scene.

Some character designs

I use C# to write scripts for the game in Unity. As a rule of thumb, I always aim to address most of the functionalities I want to implement in code, as opposed to in the Unity editor, as that could complicate things and make the game scenes more dependent on manual assignments of connections between gameObjects and components.

Currently, I have the general outline of the game's world view and story put down, and I'm working on becoming better at filling details & writing smaller filler stories. Additionally, I'm learning digital drawing and music composition to create the art in the game.
